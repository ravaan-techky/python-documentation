## Python

###Boolean datatype:
- value should be 'True' and 'False'.
- case sensitive.

  Example, - 
	```markdown
	Input=>
		my_result = true
	Output=>
		compilation failure.

	Input=>
		my_result = True
		my_result
	Output=>
		True
	```
